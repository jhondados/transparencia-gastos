# 🏛️ Transparência de Gastos Públicos

> Dashboard de análise de gastos públicos com IA. Detecta anomalias, compara órgãos e torna o orçamento público compreensível para o cidadão.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/transparencia-gastos/

[![Vertical](https://img.shields.io/badge/Vertical-Governança-blue)]() [![IA](https://img.shields.io/badge/IA-ML-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**312** Órgãos · **R$ 2.1T** Gastos · **1.847** Anomalias · **R$ 340M** Economia pot.

## 🧠 Tecnologias de IA
ML · Anomaly Detection · Data Viz

## ✨ O que faz
- Análise de gastos por órgão
- Detecção de anomalias
- Comparação histórica
- Transparência cidadã

## 🗄️ Fonte de dados real
[Portal da Transparência](https://portaldatransparencia.gov.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com Portal da Transparência
2. **Processamento com IA** — ML · Anomaly Detection · Data Viz
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
